## George Mastorakos

*Health AI, built to be trusted.*

I build and validate AI for health, and I borrow heavily from how living systems already solve the same problems. Physician by training (Mayo MD, biomimicry MS); most of my time now goes to agents, clinical ML tooling, and evaluation: the unglamorous work that decides whether any of this is safe to act on.

### What I'm building

- **[Atlas](https://github.com/george-masto/atlas-template)** is an always-on Claude Code agent that lives on your Mac and answers from your phone over Telegram. No API key, nothing leaves the machine.
- **Health Agent Rounds** sends a health AI agent "on rounds," an evaluation suite that puts clinical agents through realistic cases before real patients do.
- **MLRx** is clinical ML prototyping with rigor by construction: leakage-safe pipelines, the right metrics, Model Facts labels.
- **[The Long Listen](https://georgemasto.com/long_listen.html)** is an interactive essay on reading the body's quiet, ambient signals.

I also advise **[EpiFocus](https://www.epifocus.org/)** (seizure prediction from brain connectivity) and the **[Pitt HexAI Lab](https://pitthexai.github.io/)**. Earlier: FDA-cleared neuroimaging at [Cortechs.ai](https://www.cortechs.ai), and clinical validation and safety at Scienza Health.
